# Artsense 🖼️

> 台灣公共藝術指紋庫 - 杜絕抄襲，守护原创

## 專案簡介

Artsense 是台灣首個「公共藝術指紋庫」，透過 AI 影像比對技術，幫助公共藝術審查委員在決標前發現潛在抄襲作品。

## 核心功能

- ✅ **影像指紋建立**：自動爬取全台公共藝術作品，建立向量指紋庫
- ✅ **相似度比對**：雙軌比對系統（視覺 + 概念）
- ✅ **PDF 報告生成**：一鍵生成審查報告
- ✅ **API 接口**：支援系統整合

## 技術棧

- **影像處理**：DINOv2 + CLIP
- **向量資料庫**：ChromaDB
- **爬蟲**：Scrapling
- **後端**：FastAPI
- **部署**：Docker

## 專案結構

```
Artsense/
├── src/              # 原始碼
│   ├── crawler/      # 爬蟲模組
│   ├── api/          # FastAPI 後端
│   ├── compare/      # 比對演算法
│   └── report/       # 報告生成
├── data/             # 資料目錄
│   ├── raw/          # 原始圖片
│   └── processed/    # 處理後資料
├── models/           # 模型檔案
├── reports/          # 生成報告
└── docs/             # 文件
    ├── 工作日誌.md    # 工作記錄
    └── 成本紀錄.md   # 成本追蹤
```
