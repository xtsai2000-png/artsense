### 2026-03-24 (二) - 相似度比對頁面 + SSE Bug 修復

**已完成：**
- ✅ `/compare` 相似度比對頁面（4步驟流程）
  - Step 1：上傳圖片
  - Step 2：rembg 自動去背 + 預覽
  - Step 3：人工審查去背結果
  - Step 4：SSE 即時進度 + 結果展示
- ✅ `/api/compare/upload` 上傳 + 去背 API
- ✅ `/api/compare/image/{id}` 取得處理後圖片
- ✅ `/api/compare/search/{id}` 向量相似度搜尋（SSE 串流）
- ✅ GitHub Push

**Bug 修復：**
- **SSE 完成事件漏接問題**：`approveAndSearch()` 中，Stream 最後一個 chunk `done=true` 但 `value` 為空，導致 `decoder.decode()` 未被呼叫，buffer 裡最後一筆 `complete` 事件來不及處理就 break 迴圈
- **解法**：重構 SSE 處理邏輯，`done=true` 時先 `decoder.decode()` flush 殘留資料，再處理 buffer
